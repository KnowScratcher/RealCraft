# RealCraft
### RealCraft 是一個Minecraft的仿真合成插件，我們也將一些Mojang不讓我們合成的東西加入了合成方法，提高玩家的合成意願，也讓生存更加方便!
## Realcraft六大優點
- ✔ **直覺合成**
- ✔ **多元合成**
- ✔ **趣味合成**
- ❌ **不嚴重破壞遊戲平衡**
- ❌ **不需要設定，內建設定包辦一切**
- ❌ **不需要太多空間，1.38MB占用小**
